# Dhanya Sukanth B.K. — Portfolio

Personal portfolio website for **Dhanya Sukanth B.K.**, Senior Cloud Engineer at Microland (Bengaluru, India), supporting GE's global infrastructure across Azure and AWS.

## Tech Stack

- Vanilla HTML5 / CSS3 / JavaScript (no frameworks)
- Google Fonts: Clash Display, Cabinet Grotesk, JetBrains Mono
- Fully self-hosted — no external JS dependencies

## Structure

```
portfolio/
├── index.html          # Main entry point (lean shell, ~94 KB)
├── assets/
│   ├── css/style.css   # All styles
│   └── js/
│       ├── main.js     # Core interactions (stars, nav, animations, stats)
│       └── chatbot.js  # Portfolio assistant chatbot
```

## Features

- Animated star field loader terminal
- Scroll-reveal animations
- Interactive stats counters
- Card tilt effects
- Keyword-matching portfolio chatbot
- Security headers (CSP, Referrer-Policy, Permissions-Policy)
- No tracking, no Cloudflare proxies
