Senior Engineer — Cloud
Microland Pvt. Ltd · Bengaluru, India
MAR 2024 — PRESENT
  • Managed and maintained GE MyApps global infrastructure across production and stage environments; built and maintained monitoring stack including OpenSearch, Alert Manager, and Grafana.
  • Regularly checked server health, handled server maintenance (patching, updates, and security fixes), and optimized performance by cleaning up unused resources via disk cleanup on production and stage environments.
  • Provisioned and managed cloud infrastructure on Azure and AWS using Terraform — maintaining state files, writing reusable modules, and applying changes through controlled plan/apply workflows.
  • Monitored AWS resources using AWS Management Console, CloudTrail, and AWS Config; used ServiceNow to track and resolve user issues.
  • Established incident response plans, managed change processes and request items, and tracked system performance with Prometheus as the data source for Grafana dashboards.
  • Set up proactive alerts and moved alerts from Prometheus Alert Manager into Grafana dashboards for better readability and visibility.
  • Resolved technical issues with a focus on client satisfaction; maintained web proxies and VPN connections, and worked with L4 advisors to meet SLAs.
  • Troubleshot Zscaler proxy issues and managed URL classification policies (Zscaler / Private PZEN) via Policy Admin.
  • Validated DNS configurations and coordinated with DNS teams for domain setup and resolution issues.
  • Used Wireshark, Fiddler, application logs, and HAR files for debugging and network-level troubleshooting.
  • Created documentation and knowledge base articles to assist L1, L1.5, and L2 support teams.
  • Audited ServiceNow incidents, changes, and requests to ensure data accuracy and process compliance.
